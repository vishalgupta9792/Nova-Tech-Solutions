# Nova Tech Solutions Website

Premium startup landing website for **Nova Tech Solutions** built with Next.js, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Included sections

- Hero with CTA
- About
- Services
- ERP + Result Features
- Social Media Management
- Why Choose Us
- Pricing (3 plans)
- Portfolio
- Testimonials
- Contact + WhatsApp CTA
